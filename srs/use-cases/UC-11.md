# UC-11 — Manage Medicine Stock

Use Case ID: UC-11

Use Case Name:
Manage Medicine Stock

Actors:
Staff Member

Preconditions:
- Medicine inventory must exist.

Postconditions:
- Medicine stock is updated successfully.

Main Flow:
1. Staff opens inventory section.
2. Adds or updates medicine quantities.
3. Saves stock details.
4. System updates inventory records.

Alternative Flow:
- If stock quantity is invalid, system displays error.

Business Rules:
- Stock quantity cannot be negative.
