# UC-12 — Close Case

Use Case ID: UC-12

Use Case Name:
Close Case

Actors:
Supervisor Doctor

Preconditions:
- Treatment must be completed.

Postconditions:
- Case status changes to closed.

Main Flow:
1. Supervisor opens completed case.
2. Reviews final treatment records.
3. Confirms case closure.
4. System updates case status.

Alternative Flow:
- If treatment is incomplete, system prevents closure.

Business Rules:
- Only completed cases can be closed.
