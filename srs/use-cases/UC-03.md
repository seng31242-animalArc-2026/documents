# UC-03 — Create Case

Use Case ID: UC-03

Use Case Name:
Create Case

Actors:
Staff Member

Preconditions:
- Animal must be registered.

Postconditions:
- New treatment case is created.

Main Flow:
1. Staff opens case management section.
2. Selects animal profile.
3. Enters case details and symptoms.
4. Submits case information.
5. System creates case record.

Alternative Flow:
- If animal record is missing, system prevents case creation.

Business Rules:
- Every case must be linked to a registered animal.
