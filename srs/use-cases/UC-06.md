# UC-06 — Assign Volunteer Doctor

Use Case ID: UC-06

Use Case Name:
Assign Volunteer Doctor

Actors:
Supervisor Doctor

Preconditions:
- Case and volunteer doctor must exist.

Postconditions:
- Volunteer doctor is assigned successfully.

Main Flow:
1. Supervisor opens case details.
2. Selects volunteer doctor.
3. Confirms assignment.
4. System updates case assignment.

Alternative Flow:
- If volunteer doctor is unavailable, system shows warning.

Business Rules:
- Volunteer doctor must be approved by supervisor.
