# UC-13 — Send Donation

Use Case ID: UC-13

Use Case Name:
Send Donation

Actors:
Donor

Preconditions:
- Donor must provide payment information.

Postconditions:
- Donation is recorded successfully.

Main Flow:
1. Donor opens donation page.
2. Enters donation amount.
3. Provides payment details.
4. Confirms payment.
5. System records donation and sends confirmation.

Alternative Flow:
- If payment fails, system displays error message.

Business Rules:
- Donation transactions must be securely processed.
