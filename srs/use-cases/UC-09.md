# UC-09 — Update Case & Handover Notes

Use Case ID: UC-09

Use Case Name:
Update Case & Handover Notes

Actors:
Volunteer Doctor

Preconditions:
- Case must already exist.

Postconditions:
- Case updates and handover notes are saved.

Main Flow:
1. Volunteer opens case details.
2. Adds treatment updates.
3. Enters handover notes.
4. Saves changes.
5. System updates case record.

Alternative Flow:
- If notes are empty, system requests required information.

Business Rules:
- Handover notes must include latest treatment details.
