# UC-07 — Review & Approve Progress

Use Case ID: UC-07

Use Case Name:
Review & Approve Progress

Actors:
Supervisor Doctor

Preconditions:
- Treatment progress must be available.

Postconditions:
- Progress review is completed.

Main Flow:
1. Supervisor opens treatment progress.
2. Reviews notes and updates.
3. Approves or rejects progress.
4. System saves review results.

Alternative Flow:
- If progress is incomplete, supervisor requests corrections.

Business Rules:
- Only supervisors can approve progress updates.
