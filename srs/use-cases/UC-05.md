# UC-05 — Generate Report & Dashboard

Use Case ID: UC-05

Use Case Name:
Generate Report & Dashboard

Actors:
Administrator / Supervisor Doctor

Preconditions:
- System data must be available.

Postconditions:
- Reports and dashboard are generated successfully.

Main Flow:
1. User opens dashboard section.
2. Selects report type.
3. System collects required data.
4. Dashboard and reports are generated.
5. User views or downloads reports.

Alternative Flow:
- If data is unavailable, system shows empty report message.

Business Rules:
- Reports must only show authorized data.
