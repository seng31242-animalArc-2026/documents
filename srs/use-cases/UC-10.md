# UC-10 — Send Notification

Use Case ID: UC-10

Use Case Name:
Send Notification

Actors:
System

Preconditions:
- Notification event must occur.

Postconditions:
- Notification is sent successfully.

Main Flow:
1. System detects event trigger.
2. Generates notification message.
3. Sends notification to relevant users.
4. Logs notification status.

Alternative Flow:
- If sending fails, system retries notification.

Business Rules:
- Notifications must contain valid case information.
