# UC-04 — Assign Supervisor Doctor

Use Case ID: UC-04

Use Case Name:
Assign Supervisor Doctor

Actors:
Administrator

Preconditions:
- Case must already exist.

Postconditions:
- Supervisor doctor is assigned successfully.

Main Flow:
1. Administrator opens case details.
2. Selects available supervisor doctor.
3. Confirms assignment.
4. System updates case information.

Alternative Flow:
- If no doctors are available, system displays notification.

Business Rules:
- A case must have only one supervisor doctor.
