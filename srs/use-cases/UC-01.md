# UC-01 — Register Animal

Use Case ID: UC-01

Use Case Name:
Register Animal

Actors:
Animal Owner / Staff

Preconditions:
- User must have access to the system.

Postconditions:
- Animal profile is created successfully.

Main Flow:
1. User opens animal registration form.
2. Enters animal details.
3. Uploads animal information if needed.
4. Confirms registration.
5. System saves animal profile.

Alternative Flow:
- If required details are missing, system displays an error message.

Business Rules:
- Every animal must have a unique registration ID.
